В Visual Studio Code вы можете свернуть все функции и объекты с помощью сочетания клавиш. Для этого выполните следующие шаги:

1. Откройте файл, который вы хотите отредактировать.
2. Нажмите Ctrl + K, затем Ctrl + 0 (ноль) на Windows/Linux или Cmd + K, затем Cmd + 0 на macOS.

Это действие свернет все блоки кода в открытом файле. Чтобы развернуть их обратно, вы можете использовать сочетание клавиш Ctrl + K, затем Ctrl + J (или Cmd + K, затем Cmd + J на macOS).

Также можно использовать командную палитру:
1. Откройте командную палитру с помощью Ctrl + Shift + P (или Cmd + Shift + P на macOS).
2. Введите "Fold All" и выберите соответствующую команду, чтобы свернуть все блоки кода.

Эти методы помогут вам быстро управлять видимостью кода в редакторе.
