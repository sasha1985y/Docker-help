Использование:  docker rm [OPTIONS] CONTAINER [CONTAINER...]
            docker rm $(docker ps -aq)
            Извлеките один или несколько контейнеров

            Псевдонимы:
            docker container rm, docker container remove, docker rm
            docker rm $(docker ps -a -f "status=exited" -q)
            Options:
            -f, --force                 Принудительное извлечение работающего контейнера (использует SIGKILL)
            -l, --link                  Удалите указанную ссылку
            -v, --volumes               Удалите анонимные тома, связанные с контейнером
