Usage:  docker ai [OPTIONS] COMMAND

            Docker AI Agent - Ask Gordon

            Examples:
                To start a conversation thread
                $ docker ai

                Or ask a single question:
                $ docker ai "How do I run redis?"
                $ docker ai "What's Docker Scout?"

                Or run any docker command with AI assistance:
                $ docker ai run redis
                $ docker ai pull some_image
                $ docker ai build .

            Options:
                --send-files               Send files when asked to (default true)
                --send-project-structure   Send project structure when asked to
                                            (default true)
            -s, --shell-out                Run commands after a confirmation
            -C, --working-dir string       Sets the working directory for the command
                --write-files              Write files when asked to (default true)

            Management Commands:
            config      Configuration management commands

            Commands:
            feedback    Share your feedback with Docker Inc.
            version     Show the version information
