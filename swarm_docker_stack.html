Usage:  docker stack COMMAND

            Manage Swarm stacks

            Commands:
            config          Выводит окончательный конфигурационный файл после выполнения слияний и интерполяций
            deploy          Разверните новый стек или обновите существующий
            docker stack deploy -c docker-compose.yml mystack
            ls              Стеки списков
            docker stack ls
            ps              Перечислите задачи в стеке
            rm              Удалите одну или несколько стопок
            services        Перечислите службы в стеке
            docker stack services mystack
