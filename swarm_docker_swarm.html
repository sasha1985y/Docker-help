Usage:  docker swarm COMMAND

            Manage Swarm

            Commands:
            ca                Отображение и поворот корня CA
            init              Инициализировать swarm
            docker swarm init --advertise-addr <MANAGER_IPl>
            docker swarm init --advertise-addr 192.168.1.10
            join              Присоединиться к swarm как узел и/или менеджер
            docker swarm join --token 	<SWARM_TOKEN> <MANAGER_IP>:2377
            join-token        Управление токенами присоединения
            docker swarm join --token SWMTKN-1-0defghi1234jklmnop-5678qrstuvwx90yz 192.168.1.10:2377
            docker swarm join-token worker
            leave             Оставить swarm
            unlock            Unlock swarm
            unlock-key        Управление ключом разблокировки
            update            Обновите swarm
