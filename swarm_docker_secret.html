Usage:  docker secret COMMAND

            Управлять Swarm секретами

            Commands:
            create      Создайте секрет из файла или STDIN как содержание
            docker secret create db_password ./db_password.txt
            docker service create --name postgres --secret db_password -e POSTGRES_PASSWORD_FILE=/run/secrets/db_password postgres
            inspect     Отображение подробной информации об одном или нескольких секретах
            ls          Список секретов
            docker secret ls
            rm          Удалите один или несколько секретов
            docker secret rm db_password
