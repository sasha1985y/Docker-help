Usage:  docker node COMMAND

            Manage Swarm nodes

            Commands:
            demote      Понизьте статус одного или нескольких узлов в диспетчере в swarm
            inspect     Отображение подробной информации об одном или нескольких узлах
            ls          Перечислите узлы в swarm
            docker node ls
            promote     Продвиньте один или несколько узлов в качестве менеджера в swarm
            ps          Перечислите задачи, запущенные на одном или нескольких узлах, по умолчанию на текущем узле
            rm          Удалите один или несколько узлов из swarm
            update      Обновить узел
