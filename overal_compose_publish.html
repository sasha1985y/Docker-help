Usage:  docker compose publish [OPTIONS] REPOSITORY[:TAG]

            Publish compose application

            Options:
                --dry-run                 Execute command in dry run mode
                --oci-version string      OCI image/artifact specification version
                                            (automatically determined by default)
                --resolve-image-digests   Pin image tags to digests
                --with-env                Include environment variables in the
                                            published OCI artifact
            -y, --yes                     Assume "yes" as answer to all prompts
