Использование:  docker volume COMMAND

            Управление томами

            Commands:
            create                  Создайте том
            docker volume create my_volume
            docker volume create --driver local --opt o=uid=1000 --opt o=gid=1000 --opt type=tmpfs --opt
            device=tmpfs my_custom_volume
            inspect                 Отображение подробной информации по одному или нескольким томам
            docker volume inspect my_volume
            ls                      Перечислите тома
            docker volume ls
            prune                   Удаление неиспользуемых локальных томов
            docker volume prune -f
            rm                      Удалите один или несколько томов
            docker volume rm $(docker volume ls)
            docker volume rm my_volume
            update                  Обновление тома (только для кластерных томов)
